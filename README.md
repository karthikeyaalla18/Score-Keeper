Score-Keeper
Score Keeper is a lightweight, responsive web app for real-time cricket scoring. Track runs, wickets, and overs with dynamic target calculations. It features a complete keypad for legal deliveries and complex extras, an undo function, and a live ball-by-ball scorecard. Built with HTML/CSS/JS. Live at scorekeeper.online!
CAUTION: This is mainly for the Box Cricket or for Gully cricket
